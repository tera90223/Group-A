## Requirements
* python 3.x

## Install
Clone the repository:
```
git clone <this repository>
```
Change directory into repository:
```
cd <this repository>
```
## Usage
First make the script executable:
```
chmod +x FindORF.py
```
Run the script:
```
./FindORF.py -i <fasta_file> -t <table> -l <minimum_length>
```
Get help:
```
./FindORF.py --help
```
